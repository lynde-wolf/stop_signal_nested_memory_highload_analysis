# Results — March 17, 2026

## Go Trial Performance

We first examined whether working memory load affected go trial performance as a manipulation check. Accuracy and reaction time (RT) on go trials were each submitted to a one-way repeated measures ANOVA with three levels (WM load 0, 2, 4). For accuracy, Mauchly's test indicated the sphericity assumption was satisfied. Results showed a significant effect of memory load on go trial accuracy, F(2, 74) = 8.68, MSE = 0.0005, p < .001, η²p = .190. A Bayesian model comparison corroborated this finding, ΔBIC = −7.61, BF₁₀ = 0.02 (Kass & Raftery, 1995). For RT, Mauchly's test indicated sphericity was violated (W = 0.82, p = .030), so p-values were corrected using the Greenhouse-Geisser method (Greenhouse & Geisser, 1959). Results showed a significant effect of memory load on go trial RT, F(2, 74) = 5.70, MSE = 735.28, p = .008, η²p = .134, ΔBIC = −9.13, BF₁₀ = 0.01.

## Stop Signal Reaction Time (SSRT)

This experiment aimed to test if changes in memory load (0, 2, 4 letter stimuli) has an effect on response inhibition captured by the SSRT. We compare the SSRT within subjects on a simple stopping task to the SSRT during each of the 3 memory load conditions. A direct paired comparison between simple stop SSRT (M = 243.6 ms, SD = 78.8 ms) and collapsed dual-task SSRT (M = 255.3 ms, SD = 42.9 ms) showed no significant difference, t(37) = −0.997, p = .325, d = −0.162. Following the preregistered analysis plan, the full comparison was made using a one-way repeated measures ANOVA with four levels (simple stop, 0, 2, 4). Mauchly's test indicated sphericity was violated (W = 0.25, p < .001), so p-values were corrected using the Greenhouse-Geisser method (Greenhouse & Geisser, 1959). Results showed no evidence of load affecting the SSRT, F(3, 111) = 0.82, MSE = 2166.71, p = .42, η²p = .022. A Bayesian model comparison further corroborates this finding, ΔBIC = −13.55 (Kass & Raftery, 1995). Pairwise comparisons similarly showed no evidence of an effect of memory load on SSRT (all Bonferroni-corrected ps > .59; simple stop: M = 243.6 ms; WM load 0: M = 260.1 ms; WM load 2: M = 254.5 ms; WM load 4: M = 251.5 ms).

## Stop Signal Success Rate

To corroborate the SSRT findings, we examined whether the stop signal success rate differed across conditions. Descriptive statistics confirmed that success rates were virtually identical across conditions (simple stop: M = 0.52; WM load 0: M = 0.52; WM load 2: M = 0.53; WM load 4: M = 0.52), consistent with the staircase procedure maintaining success at approximately 50%. A Bayesian model comparison showed very strong evidence against condition effects on stop signal success rate, ΔBIC = −14.58 (Kass & Raftery, 1995).

## Probe Accuracy

We next examined whether working memory load affected accuracy on the probe recognition task, as a further manipulation check. The comparison was made using a one-way repeated measures ANOVA with three levels (WM load 0, 2, 4). Mauchly's test indicated the sphericity assumption was satisfied. Results showed a significant effect of memory load on probe accuracy, F(2, 74) = 12.40, MSE = 0.0036, p < .001, η²p = .251, ΔBIC = 1.18, BF₁₀ = 1.80 (Kass & Raftery, 1995). Pairwise comparisons with Bonferroni correction revealed that probe accuracy was significantly higher at WM load 0 (M = 0.952) compared to WM load 2 (M = 0.904, p = .013) and WM load 4 (M = 0.886, p < .001), and higher at WM load 2 than WM load 4 (p = .036), confirming that the memory load manipulation was effective.

## Probe Accuracy by Stop Signal Outcome

To examine whether response inhibition success was associated with memory encoding, we compared probe accuracy on successful stop, failed stop, and go trials using paired t-tests. Bayesian model comparisons were computed following the BIC approximation method (Wagenmakers, 2007; Kass & Raftery, 1995). Overall, there was no evidence that probe accuracy differed as a function of stop signal outcome: successful stops (M = 0.911) versus failed stops (M = 0.911), t(37) = 0.02, p = .986, d = 0.003, ΔBIC = −4.33, BF₁₀ = 0.11; successful stops versus go trials (M = 0.915), t(37) = −0.65, p = .517, d = −0.11, ΔBIC = −4.28, BF₁₀ = 0.12; failed stops versus go trials, t(37) = −0.47, p = .645, d = −0.08, ΔBIC = −4.27, BF₁₀ = 0.12. These null results held consistently across all three WM load levels. At WM load 0 (successful stop: M = 0.943, failed stop: M = 0.950, go: M = 0.955), all pairwise comparisons were non-significant (ps > .14; all ΔBICs < −3.98, all BF₁₀s < 0.14). At WM load 2 (successful stop: M = 0.890, failed stop: M = 0.904, go: M = 0.907), no comparisons reached significance (ps > .09; all ΔBICs < −3.81, all BF₁₀s < 0.15). At WM load 4 (successful stop: M = 0.901, failed stop: M = 0.880, go: M = 0.884), all pairwise comparisons were similarly non-significant (ps > .13; all ΔBICs < −3.76, all BF₁₀s < 0.15).

## Sequential Dependencies: Prior-Trial Probe Outcome

> ⚠️ **Note — commission + omission errors combined.** In all analyses in this section, the "probe incorrect" group (`memory_recognition_correct_trial == 0`) includes both commission errors (wrong response given, RT present) and omission errors (no response, RT is NaN). See the Prior-Trial Probe Omission section below for omission-only analyses and the commission-only supplementary analysis at the end of this section.

We conducted a series of exploratory analyses to examine whether performance on a given trial was influenced by the probe outcome on the preceding trial. All analyses compared outcomes following a t-1 probe-correct trial versus a t-1 probe-incorrect trial (combining commission and omission errors) using paired t-tests. Bayesian model comparisons were computed following the BIC approximation method (Wagenmakers, 2007; Kass & Raftery, 1995).

**Prior probe outcome and stop success.** Following a t-1 probe-incorrect trial, participants showed a higher stop success rate (M = 0.609) compared to following a t-1 probe-correct trial (M = 0.514), t(36) = −2.347, p = .025, d = −0.391, ΔBIC = 1.66, BF10 = 2.29. Breaking this down by memory load, the effect was significant at WM load 0, t(33) = −2.325, p = .026, d = −0.405, ΔBIC = 1.63, BF10 = 2.26, and WM load 4, t(32) = −2.244, p = .032, d = −0.397, ΔBIC = 1.33, BF10 = 1.94, but not at WM load 2, t(33) = −0.624, p = .537, d = −0.109, ΔBIC = −3.13, BF10 = 0.21.

**Prior probe outcome and probe RT.** Probe RT was significantly faster following a t-1 probe-correct trial than following a t-1 probe-incorrect trial, t(37) = −2.321, p = .026, d = −0.382, ΔBIC = 1.53, BF10 = 2.15. By load, this effect was significant only at WM load 2, t(37) = −2.554, p = .015, d = −0.420, ΔBIC = 2.53, BF10 = 3.55, and not at WM load 0, t(37) = −1.434, p = .160, d = −0.236, ΔBIC = −1.58, BF10 = 0.45, or WM load 4, t(35) = −1.526, p = .136, d = −0.258, ΔBIC = −1.26, BF10 = 0.53.

**Prior probe outcome and go RT.** There was no evidence that prior probe outcome affected go trial RT overall, t(37) = −0.941, p = .353, d = −0.155, ΔBIC = −2.74, BF10 = 0.25, or at any individual WM load level (WM load 0: t(35) = −0.992, p = .328; WM load 2: t(36) = 0.335, p = .740; WM load 4: t(35) = −0.645, p = .523; all ps > .32).

**Prior probe outcome and go RT (correct vs. incorrect go responses).** As a follow-up to the overall go RT analysis, we attempted to disambiguate whether the null go RT effect differed across correct versus incorrect go responses following a probe-incorrect trial. This analysis required participants to have go trials following probe-incorrect trials in both the correct-go and incorrect-go categories. Due to the rarity of probe-incorrect trials combined with go commission errors, only 3 participants had sufficient data for this comparison (go RT correct: t(2) = −0.965, p = .341, d = −0.159; go RT incorrect: t(2) = 0.397, p = .696, d = 0.094). This analysis was therefore underpowered and the results are not interpretable.

**Prior probe outcome and SSD.** Prior probe outcome did not significantly predict stop-signal delay (SSD) on the following stop trial, t(36) = 0.651, p = .519, d = 0.109, ΔBIC = −3.18, BF₁₀ = 0.20, and this null result was consistent across WM load levels (WM load 0: t(33) = −1.034, p = .309; WM load 2: t(33) = 1.199, p = .239; WM load 4: t(32) = −0.211, p = .834; all ps > .23).

**Prior probe outcome and go accuracy.** There was no significant overall effect of prior probe outcome on go trial accuracy, t(37) = 1.726, p = .093, d = 0.284, ΔBIC = −0.69, BF10 = 0.71. At WM load 4, the effect was marginally significant but inconclusive, t(37) = 2.084, p = .044, d = 0.343, ΔBIC = 0.58, BF10 = 1.34; effects were non-significant at WM load 0, t(37) = 1.222, p = .230, and WM load 2, t(37) = 1.193, p = .240.

**Prior probe outcome and subsequent probe accuracy.** Prior probe outcome significantly predicted accuracy on the next probe trial. Following a t-1 probe-incorrect trial, subsequent probe accuracy was lower (M = 0.850) than following a t-1 probe-correct trial (M = 0.926), t(37) = 2.748, p = .009, d = 0.452, ΔBIC = 3.42, BF10 = 5.53. This effect was consistent across all WM load levels: WM load 0, t(37) = 2.483, p = .018, d = 0.408, ΔBIC = 2.22, BF10 = 3.03; WM load 2, t(37) = 2.768, p = .009, d = 0.455, ΔBIC = 3.51, BF10 = 5.80; WM load 4, t(35) = 2.519, p = .017, d = 0.426, ΔBIC = 2.41, BF10 = 3.34.

**Trial-level linear mixed model: prior probe outcome predicting stop success.** To complement the subject-level paired t-test, we fit a trial-level linear mixed effects model (LMM) predicting stop success from prior-trial probe outcome (correct vs. incorrect, combining commission and omission errors), with participant as a random intercept. The model was estimated using LBFGS optimization (N = 5,302 stop trials, N = 38 subjects; N = 438 trials following a probe-incorrect trial and 4,864 following probe-correct). Prior-trial probe outcome significantly predicted stop success at the trial level: β = −0.110, SE = 0.026, z = −4.27, p < .001, 95% CI [−0.161, −0.060]. This indicates that following a probe-incorrect trial (versus a probe-correct trial), stop success was approximately 11 percentage points higher on average. Note that the random effects covariance was estimated at the boundary (singular), suggesting that between-subject variance in the association was minimal; the fixed effect estimate should be interpreted with this in mind.

**Prior probe commission error and stop success (commission only).** A supplementary analysis restricted to commission errors only (excluding probe omissions) examined whether t-1 probe commission errors (incorrect responses with a response time present) versus t-1 probe correct responses predicted stop success on the following stop trial. N = 37 participants had at least one commission error trial, with an average of 7.7 qualifying stop trials per subject following a commission error (range: 1–23). Overall, stop success rate following a commission error (M = 0.581) was numerically higher than following a correct probe response (M = 0.514), but this difference did not reach significance, t(36) = −1.413, p = .166, d = −0.235, ΔBIC = −1.61, BF₁₀ = 0.45. No significant effect was found at any individual WM load level: WM load 0, t(32) = −1.451, p = .157, d = −0.256, ΔBIC = −1.39, BF₁₀ = 0.50 (after commission: M = 0.606; after correct: M = 0.507); WM load 2, t(31) = −0.133, p = .895, d = −0.024, ΔBIC = −3.45, BF₁₀ = 0.18 (after commission: M = 0.535; after correct: M = 0.525); WM load 4, t(30) = −0.629, p = .534, d = −0.115, ΔBIC = −3.03, BF₁₀ = 0.22 (after commission: M = 0.557; after correct: M = 0.507). Notably, the commission-only effect (d = −0.235) was substantially smaller than the combined commission+omission effect reported above (d = −0.391), suggesting the overall "probe incorrect" effect on stop success was driven primarily by omission trials rather than commission errors.

## Sequential Dependencies: Prior-Trial Probe Omission

We conducted a parallel set of analyses examining sequential effects of probe omissions (trials where participants failed to respond to the probe) on subsequent trial performance. These analyses were conducted in a subset of participants who had probe omission trials (ns range from 25 to 32 depending on the analysis). Bayesian model comparisons were computed following the BIC approximation method (Wagenmakers, 2007; Kass & Raftery, 1995).

**Probe omission and subsequent stop success.** Following a trial on which participants omitted a probe response, stop success rate on the next stop trial was substantially higher (M = 0.696) compared to following a trial with a probe response (M = 0.516), t(24) = 2.978, p = .007, d = 0.608, ΔBIC = 4.64, BF10 = 10.19. This pattern did not reach significance within individual WM load levels: WM load 0, t(13) = 2.045, p = .062, d = 0.567; WM load 2, t(16) = 1.258, p = .226, d = 0.315; WM load 4, t(19) = 1.513, p = .147, d = 0.347.

**Probe omission and subsequent go omission.** Following a probe omission trial, the rate of go trial omissions was higher (M = 0.116) compared to following a probe response trial (M = 0.022), t(29) = 2.115, p = .043, d = 0.393, ΔBIC = 0.90, BF10 = 1.57. By WM load, the effect was significant at WM load 0, t(23) = 2.344, p = .028, d = 0.489, marginal at WM load 4, t(20) = 2.033, p = .056, d = 0.455, and non-significant at WM load 2, t(23) = 1.607, p = .122, d = 0.335.

**Probe omission and subsequent go RT.** There was no evidence that probe omission on the prior trial affected go RT on the following trial, t(29) = 0.441, p = .662, d = 0.082, ΔBIC = −3.20, BF10 = 0.20 (omission: M = 753 ms; response: M = 744 ms). This null result held across all WM load levels (WM load 0: t(22) = 1.169, p = .255; WM load 2: t(23) = −0.391, p = .700; WM load 4: t(18) = 0.612, p = .548; all ps > .25).

**Probe omission and subsequent SSD.** Prior probe omission did not significantly predict SSD on the following stop trial, t(24) = 0.085, p = .933, d = 0.017, ΔBIC = −3.21, BF10 = 0.20 (omission: M = 467 ms; response: M = 466 ms). This null result was consistent across WM load levels (WM load 0: t(13) = −0.092, p = .928; WM load 2: t(16) = 0.260, p = .798; WM load 4: t(19) = 1.281, p = .216; all ps > .21).

**Probe omission and subsequent probe omission.** Prior probe omission significantly predicted probe omission on the next probe trial (M = 0.135 vs. M = 0.024), t(31) = 2.569, p = .015, d = 0.461, ΔBIC = 2.71, BF10 = 3.88. This sequential clustering of omissions was consistent across all WM load levels: WM load 0, t(27) = 2.569, p = .016, d = 0.494, ΔBIC = 2.79, BF10 = 4.04; WM load 2, t(26) = 2.468, p = .021, d = 0.484, ΔBIC = 2.39, BF10 = 3.30; WM load 4, t(23) = 2.477, p = .021, d = 0.516, ΔBIC = 2.50, BF10 = 3.49.

**Probe omission and subsequent probe RT.** Prior probe omission significantly predicted probe RT on the next probe trial (analyses restricted to trials where the participant responded on trial t). Following an omission, probe RT was faster (M = 728 ms) than following a probe response (M = 803 ms), t(31) = 2.517, p = .017, d = 0.445, ΔBIC = 2.49, BF₁₀ = 3.47. This effect was significant at WM load 0, t(26) = 2.552, p = .017, d = 0.491, ΔBIC = 2.74, BF₁₀ = 3.94 (after omission: M = 583 ms; after response: M = 680 ms), and WM load 4, t(21) = 2.774, p = .011, d = 0.591, ΔBIC = 3.78, BF₁₀ = 6.62 (after omission: M = 818 ms; after response: M = 951 ms), with a marginal trend at WM load 2, t(26) = 1.988, p = .058, d = 0.383, ΔBIC = 0.52, BF₁₀ = 1.30 (after omission: M = 785 ms; after response: M = 871 ms).

**Probe omission and subsequent go accuracy.** There was no significant overall effect of prior probe omission on go trial accuracy, t(29) = −1.576, p = .126, d = −0.288, ΔBIC = −0.94, BF₁₀ = 0.63, though the numerically higher accuracy following omissions (M = 0.939) versus probe responses (M = 0.869) suggests a possible speed-accuracy tradeoff consistent with the go omission and go RT patterns. No individual WM load reached significance (WM load 0: t(23) = −1.956, p = .063; WM load 2: t(23) = −1.377, p = .182; WM load 4: t(20) = −1.744, p = .097; all ps > .06).

**Probe omission and subsequent probe commission rate.** Prior probe omission did not predict the commission error rate on the next probe trial overall, t(31) = 0.756, p = .455, d = 0.134, ΔBIC = −2.88, BF₁₀ = 0.24 (after omission: M = 0.061; after response: M = 0.079), nor at any individual WM load level (all ps > .21).

## Sequential Dependencies: Prior-Trial Probe Commission Error

We conducted an additional set of analyses examining sequential effects of probe commission errors (trials where participants responded incorrectly to the probe with a response time present) on subsequent trial performance, complementing the omission-specific analyses above. Omission trials (no response) were excluded from the prior-trial IV in all analyses here. All analyses used paired t-tests comparing outcomes following a commission error versus following a correct probe response. Bayesian model comparisons were computed following the BIC approximation method (Wagenmakers, 2007; Kass & Raftery, 1995).

**Prior commission error and subsequent stop success.** As reported in the prior section, stop success following a commission error (M = 0.581) was numerically but not significantly higher than following a correct probe response (M = 0.514), t(36) = −1.413, p = .166, d = −0.235, ΔBIC = −1.61, BF₁₀ = 0.45, with no significant effects at any individual WM load (all ps > .15).

**Prior commission error and subsequent probe RT.** There was no significant overall effect of prior commission error on probe RT, t(37) = 1.274, p = .211, d = 0.207, ΔBIC = −2.01, BF₁₀ = 0.37 (after correct: M = 716 ms; after commission: M = 700 ms). A significant effect emerged at WM load 2 only, t(37) = 2.102, p = .042, d = 0.341, ΔBIC = 0.65, BF₁₀ = 1.38 (after correct: M = 789 ms; after commission: M = 745 ms), indicating faster probe RT following a commission error at the intermediate load level. No effects at WM load 0, t(36) = −0.312, p = .757, ΔBIC = −3.51, BF₁₀ = 0.17, or WM load 4, t(35) = 0.584, p = .563, ΔBIC = −3.23, BF₁₀ = 0.20.

**Prior commission error and subsequent go RT.** Prior commission error did not predict go RT on the following trial, t(36) = 1.071, p = .291, d = 0.176, ΔBIC = −2.45, BF₁₀ = 0.29, nor at any individual WM load level (WM load 0: t(34) = 1.297, p = .203; WM load 2: t(35) = −0.405, p = .688; WM load 4: t(33) = 0.689, p = .496; all ps > .20).

**Prior commission error and subsequent SSD.** Prior commission error did not predict SSD on the following stop trial, t(36) = −0.737, p = .466, d = −0.121, ΔBIC = −3.06, BF₁₀ = 0.22, nor at any WM load level (all ps > .16).

**Prior commission error and subsequent go accuracy.** Prior commission error did not predict go accuracy on the following trial, t(36) = −0.845, p = .403, d = −0.139, ΔBIC = −2.88, BF₁₀ = 0.24, nor at any WM load level (all ps > .40).

**Prior commission error and subsequent go omission rate.** Overall, prior commission error did not predict go omission rate, t(36) = 0.095, p = .925, d = 0.016, ΔBIC = −3.60, BF₁₀ = 0.17. However, a significant effect emerged at WM load 0: following a commission error at WM load 0, go omission rate was higher (M = 0.022) than following a correct response (M = 0.004), t(34) = −3.312, p = .002, d = −0.560, ΔBIC = 6.23, BF₁₀ = 22.53. No significant effects at WM load 2, t(35) = 0.739, p = .465, or WM load 4, t(33) = 0.923, p = .363. The WM load 0 effect should be interpreted with caution given the overall null result and absence of a clear theoretical account.

**Prior commission error and subsequent probe omission rate.** Prior commission error did not significantly predict probe omission rate on the next probe trial overall, t(37) = 1.656, p = .106, d = 0.269, ΔBIC = −0.92, BF₁₀ = 0.63 (after correct: M = 0.035; after commission: M = 0.019). A significant effect emerged at WM load 4 only, t(35) = 2.314, p = .027, d = 0.386, ΔBIC = 1.54, BF₁₀ = 2.16, where probe omission rate was lower following a commission error (M = 0.016) than following a correct response (M = 0.046). No effects at WM load 0, t(36) = 1.177, p = .247, or WM load 2, t(37) = 0.296, p = .769.

**Prior commission error and subsequent probe commission rate.** Prior commission error significantly predicted commission error rate on the next probe trial overall, t(37) = 2.148, p = .038, d = 0.348, ΔBIC = 0.83, BF₁₀ = 1.51: commission rate was lower following a prior commission error (M = 0.054) than following a correct response (M = 0.078). This pattern — fewer commission errors after a commission error — is consistent with post-error correction or increased response caution. By WM load, the effect was significant at WM load 2, t(37) = 2.086, p = .044, d = 0.338, ΔBIC = 0.59, BF₁₀ = 1.34 (after correct: M = 0.105; after commission: M = 0.061), and non-significant at WM load 0, t(36) = 1.252, p = .219, and WM load 4, t(35) = 0.662, p = .512.
